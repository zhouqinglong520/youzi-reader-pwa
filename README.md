# 游资心法 · 实战笔记 PWA

📖 **130 份顶级游资实战笔记** · 26 位游资心法学习笔记 · 离线阅读

## 🌐 在线访问

**PWA 地址**：https://zhouqinglong520.github.io/youzi-reader-pwa/

支持手机/平板/电脑浏览器访问，可添加到主屏幕作为 App 使用。

## 📱 功能特性

- ✅ **130 份笔记**：68 份 P0/P1/P2 实战笔记 + 62 份 C 盘 26 位游资学习笔记
- ✅ **8 大分类**：高手手记 / 顶级游资心法 / 经典思维 / 集合竞价 / 综合资料 / 打板追涨 / 心态方法 / 案例复盘
- ✅ **离线阅读**：PWA 技术支持，首次加载后可离线使用
- ✅ **全文搜索**：支持标题、内容、标签搜索
- ✅ **三栏布局**：分类栏 / 笔记列表 / 详情阅读
- ✅ **移动端适配**：响应式设计，手机/平板/电脑完美适配
- ✅ **Markdown 渲染**：完整支持 Markdown 语法

## 📚 内容来源

### P0/P1/P2 阶段（68 份）
- D 盘"股市游资综合资料合集"中 68 份实战类 PDF/DOCX
- 包含：养家心法、Asking 理念、赵老哥、集合竞价奥义等经典内容

- 包含：92 科比、ASKING、丁一熊、万法归宗、乔帮主、令胡冲、凡倍无名、孤独牛背、小鳄鱼、徐翔、明王、榜中榜、欢乐海岸、浓汤野人、涅盘重升等
- 交易体系整合版、模拟交易系统等

## 🛠 技术栈

- **React 19** + **TypeScript**
- **Vite 8** 构建
- **PWA** 离线支持（Service Worker）
- **Markdown 渲染器**：自研轻量级渲染器
- **GitHub Pages** 部署

## 📦 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 🚀 部署

项目使用 GitHub Pages 自动部署，push 到 `main` 分支即可。
