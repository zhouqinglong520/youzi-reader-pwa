# 游资心法 · 实战笔记

130 份顶级游资实战笔记，PWA 离线阅读应用。

## 快速开始

**在线访问：** https://zhouqinglong520.github.io/youzi-reader-pwa/

- 手机浏览器打开后，点击"添加到主屏幕"即可像 App 一样使用
- 支持离线阅读，首次加载后无需网络
- 支持收藏、阅读进度记忆、深色模式、字体调节

## 功能特性

### 阅读体验
- 📖 **130 份笔记** — 覆盖 25+ 位知名游资的实战心得
- 🔍 **全文搜索** — 快速定位任意关键词
- 📑 **主题分类** — 按游资名字/主题筛选
- 🌙 **深色模式** — 自动跟随系统 + 手动切换
- 🔤 **字体调节** — 3 档字号自由切换
- 📊 **阅读进度** — 自动记忆上次阅读位置

### 收藏与管理
- ⭐ **收藏笔记** — 收藏重要内容，数据本地存储
- 📈 **进度追踪** — 每篇笔记显示阅读进度百分比

### PDF 导出
- 📥 **PDF 导出** — 单篇笔记可导出 PDF 离线保存

### 离线能力
- 💾 **PWA 支持** — Service Worker 缓存，断网可用
- 📱 **安装到桌面** — 像原生 App 一样体验

## 技术栈

- Vite 6 + React 19 + TypeScript
- 纯前端无后端，数据内置
- PWA + Service Worker 离线缓存
- jsPDF 本地 PDF 生成
- GitHub Pages 静态托管
